pub struct Stack(_, _);

The Stack widget is a container which only shows one of its children at a time. In contrast to Notebook, Stack does not provide a means for users to change the visible child. Instead, the StackSwitcher widget can be used with Stack to provide this functionality.

Transitions between pages can be animated as slides or fades. This can be controlled with Stack::set_transition_type. These animations respect the Settings:gtk-enable-animations setting.

The Stack widget was added in GTK+ 3.10.

fn set_homogeneous(&self, homogeneous: bool)

Sets the Stack to be homogeneous or not. If it is homogeneous, the Stack will request the same size for all its children. If it isn't, the stack may change size when a different child becomes visible.

Since 3.16, homogeneity can be controlled separately for horizontal and vertical size, with the Stack:hhomogeneous and Stack:vhomogeneous.
