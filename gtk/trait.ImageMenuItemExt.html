fn set_accel_group(&self, accel_group: &AccelGroup)

Specifies an accel_group to add the menu items accelerator to (this only applies to stock items so a stock item must already be set, make sure to call ImageMenuItemExt::set_use_stock and MenuItemExt::set_label with a valid stock item first).

If you want this menu item to have changeable accelerators then you shouldnt need this (see ImageMenuItem::new_from_stock).
